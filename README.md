# 五子棋游戏 (Gomoku Game)

一个使用现代前端技术栈构建的精美五子棋游戏。

## 🎮 游戏特性

- **标准15×15棋盘**: 采用传统五子棋规格
- **黑白交替下棋**: 自动切换玩家回合
- **智能胜负判断**: 实时检测横、竖、斜方向的五子连线
- **游戏重置功能**: 一键重新开始新游戏
- **现代化UI设计**: 美观的界面和流畅的动画效果
- **响应式布局**: 完美适配桌面和移动设备
- **最后一步标记**: 高亮显示最新落子位置

## 🛠️ 技术栈

- **React 18** - 现代化前端框架
- **TypeScript** - 类型安全的JavaScript
- **Vite** - 快速的构建工具
- **CSS3** - 现代样式和动画

## 📦 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

4. 预览生产版本：
```bash
npm run preview
```

## 🎯 游戏规则

1. 黑棋先行，双方轮流下棋
2. 在棋盘交叉点上放置棋子
3. 先连成五子（横、竖、斜任意方向）的一方获胜
4. 棋盘填满无人获胜则为平局
